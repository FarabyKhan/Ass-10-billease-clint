  Project Name: billease - A Utility Bill Management System

  Projects Purpose:

     It's a learning project base vite + react project with Firebase authentication and Database Management System (MongoDB) that denots how real API are made and used in a react project so that the user can brows the website with real time updated data of security with proper authorization.

    Key Features: 
- React Router for route management and navigation
- Real Clint-Server API  with MongoDB.
- Middleware from Firebase ID Token for user Authorization.
- Firebase Authentication (Email/Password & Google Sign-in).
- Protected routes for authorized users.
- React Hot Toast and Sweet Alert for real-time notifications.
- Smooth animations with Swiper.js and Lottie React.
- Can Download the users data in PDF formate.
- Live API using Vercel. 
- Responsive UI using Tailwind CSS and DaisyUI.
- Clean and modern design focused on user experience.
